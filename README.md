# Digital Clock & Stopwatch

A simple and responsive Digital Clock & Stopwatch built using HTML, CSS, and JavaScript.

## Features

- Live Digital Clock
- Stopwatch Functionality
- Start Button
- Stop Button
- Reset Button
- Responsive Design
- Modern User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```
Digital-Clock-Stopwatch/
│
├── index.html
├── clock.css
└── clock.js
```

## How to Run

1. Download the project files.
2. Open `index.html` in your browser.
3. Enjoy the Digital Clock and Stopwatch.
